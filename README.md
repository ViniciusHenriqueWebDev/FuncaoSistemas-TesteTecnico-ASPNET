# Projeto de teste técnico para a empresa Função Sistemas: 

## Objetivo: cadastro de clientes e beneficiários, desenvolvido em ASP.NET MVC

## Descrição

Este sistema permite o cadastro de clientes e a gestão de beneficiários associados a cada cliente. O objetivo é demonstrar habilidades em desenvolvimento web, validação de dados, manipulação dinâmica de listas e integração frontend-backend.

## Funcionalidades

- Cadastro, edição e exclusão de clientes.
- Cadastro, edição e exclusão de beneficiários para cada cliente.
- Validação de CPF (formato e dígitos) para clientes e beneficiários.
- Prevenção de CPFs duplicados entre beneficiários do mesmo cliente.
- Máscara automática para campos de CPF.
- Feedback visual para operações de sucesso e erro.
- Atualização dinâmica do grid de beneficiários.
- Envio dos dados dos beneficiários em JSON para o backend.
- Campos obrigatórios validados via HTML5 (`required`).

## Tecnologias Utilizadas

- ASP.NET Framework 4.8 (MVC)
- C#
- JavaScript (jQuery)
- HTML5 e CSS3
- Bootstrap
